¿Quienes Somos?


Somos cuatro estudiantes del Colegio Pio IX de Buenos Aires, que mediante un proyecto tecnologico buscamos emprender realizando dispositivos electronicos avanzados.
Mediante este proyecto buscamos facilitar el montaje de una linea de produccion en comercios y restaurantes.

El Proyecto


La idea fundamental del proyecto surgío de un analisís a los procesos efectuados a la hora de realizar un pedido gastronomico en restaurantes o comercios de comida.
Observamos que la produccion de bebidas generaba un retraso en la entrega del pedido.
Para solucionar esto, planteamos una aplicación mediante la cual se pueda pedir una bebida para que una maquina dispnsadora ejecute esa orden y genere el pedido.

A continuación, en detalle realizamos una idea de fuerza la cual constituía ciertos aspectos de la maquina y el proyecto en sí.

Circuitos Utilizados


Dentro de este proyecto tecnologico utilizamos varias plaquetas destinadas a darle un correcto funcionamiento a la maquina dispensadora.

Shield KL25

Plaqueta de interconexión entre microcontrolador y plaquetas. Contiene una etapa de potencia para regular corrientes y voltajes.

Conector Display LCD

Plaqueta de interconexión entre display y bornes de conexión.

Controlador Servo Motores

Plaqueta de control para servo motores. Aumenta la tension de la señal del microcontrolador a la requerida para mover el servo motor.

Conector Bombas de Agua

Plaqueta de interconexión con microcontrolador para bombas de agua de 5v.

Regulador 7805

Plaqueta de rectificación de corriente para voltaje de 5v.

Controlador de Sensores

Plaqueta de interconexión de sensores infrarojos.

Teclado Mecanico

Plaqueta de teclado mecanico matricial utilizada para interfaz usuario fisica.

Controlador RGB

Plaqueta controlador de RGB para teclado mecanico.

Aplicacion Web


La aplicación web es la encargada de gestionar los pedidos desde un dispotivo por parte del empleado a cargo.
Dentro de esta aplicacion tenemos la posibilidad de pedir una de las 4 tipos de bebidas.
La aplicacion esta conectada a traves de un modulo wifi el cual se conecta directamente a la red correspondiente para la gestion de los pedidos.
La aplicacion se compone de una tabla la cual almacena los datos de los pedidos, y una interfaz de selección de pedidos.
